# plusコマンド
![test](https://github.com/ahaya8810/robosys2022/actions/workflows/test.yml/badge.svg)

[使用用途]

標準入力から読み込んだ数字を足した結果を表示する。

[起動する手順]

 seq 〇　| ./plus (〇には任意の数字）　を入力する。

[起動例]

 seq 5 | ./plus

 15.0 

## 必要なソフトウェア
* Python
  * テスト済み: 3.7〜3.10

## テスト環境
* Ubuntu
